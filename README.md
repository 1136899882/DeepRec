# DeepRec
In this repository, we implement many recent deep learning based recommender models with Tensorflow.

# Requirements
* Tensorflow 1.7+
* Python 3.5+
* numpy
* scipy
* sklearn
* pandas

# Implemented Algorithms
* AutoRec (www)
* CDAE (WSDM)
* NeuMF (WWW)
* CML (WWW)
* etc.
